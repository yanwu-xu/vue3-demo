# Attributes

|   参数   |   说明   |   类型   |   默认值   |
| ---- | ---- | ---- | ---- |
|   icon-class   |   必填，icon类名   |   String   |   —   |
|   color   |   icon填充颜色   |   String   |   #fff   |
|   class-name   |   自定义类名   |   String   |   —   |
|   active   |   true则标识鼠标样式为手型   |   Boolean   |   false   |
|   desc   |   气泡框的内容，有值才显示气泡框   |   String   |   —   |
